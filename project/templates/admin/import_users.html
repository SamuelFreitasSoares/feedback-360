Importar Usuários a partir de Arquivo CSV

{% csrf_token %}
Selecione um arquivo CSV com os dados dos usuários.
Formato do Arquivo

O arquivo CSV deve seguir o formato específico para cada tipo de usuário:

Alunos: nome,email,matricula,curso_id

Professores: nome,email

Coordenadores: nome,email,curso_id

A primeira linha do arquivo será ignorada (cabeçalhos). A senha padrão para todos os usuários importados será "123456".

Instruções

  1. Selecione o tipo de usuário que deseja importar
  2. Prepare um arquivo CSV com os dados no formato especificado
  3. Faça upload do arquivo
  4. Verifique os resultados da importação
Certifique-se de que:
  • Os IDs de curso existam no sistema
  • Os emails sejam únicos
  • As matrículas de alunos sejam únicas
